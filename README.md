
# autarchy-webapp

## Install

```
git clone https://github.com/hacknlove/autarchy-webapp.git
cd autarchy-webapp
npm i
npm run build
```

## Start

```
npm start
```

It uses the environmental variable `MONGO_URL` which defaults to `mongodb://localhost:27017/autarchy`
